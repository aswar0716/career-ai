# Career AI

An intelligent career management system that acts as a personal job search assistant.

## Core Capabilities
- Application tracking
- Resume and cover letter optimization
- Skill gap analysis
- Email automation
- Career analytics

## Tech Stack
- Frontend: Next.js, TypeScript
- Backend: Node.js, TypeScript
- AI Services: Python
- Database: PostgreSQL or MongoDB
- AI APIs: OpenAI, Claude
